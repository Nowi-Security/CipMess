# CipMess?

CipMess is an app where you can encrypt messages and export files with that same message to later import the file and decrypt that message.


# Technologies used
 - Languages: Kotlin
 - Tools: AES256/GCM, AndroidKeyStore, security, Biometric and Json 
 - Library (Android)
